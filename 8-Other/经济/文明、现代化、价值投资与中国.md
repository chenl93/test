### 老问题与新史学

  - 为什么中国与西方差距如此悬殊？
  - 中国如何能够赶上西方？
  - 赶上以后的中国会是什么样的？是否还能重现往日的辉煌？

#### 东西方的差距
- 是过去200多年的时间产生的目前西方的领先和东方的衰落
  + 中国是王朝更替，君主专制；科举制选拔人才；
- 李鸿章言：中国在1840年面临的是一场“三千年未有之变局”
  + 李鸿章的传记还没有读，只是在曾国藩的书中读到过。他的功过也应该比较复杂，但他说出的这句话证明他对当时的判断还是比较深刻的
- 所有的人类女性都可以追溯到一个共同的祖先，她居住在非洲，诞生于大约20万年以前；后面又找到了人类共同的男性祖先
  + 那所有的人类都是有血缘关系的，经过不断的繁衍进化。就像每个小家庭的不断分裂，随着不断的分裂，关系与血缘的浓度不断下降
  + 今天的人类都起源于同一个祖先
  + 由于人类都起源于一个祖先，所以不能以传统观点种族，文化的不同来解释东西方领先理论
- 新史学：利用科技碳14、基因等各个领域科学前沿发展，跨学科重新构造解读历史
  + 历史的验证，研究也要依赖跨学科，这里提起的跨学科想起芒格
  + 不局限于文字史


### 文明的轨迹
- 社会发展指数：即一个社会能够办成事的能力；想要衡量社会发展的程度，最重要的指数就是摄取能量和使用能量的能力。
- 能量守恒原理：一个社会要能办成事，需要摄取和使用能量的能力
- 能量（最具代表性的人类摄取和使用能量的四个方面）：
  - 摄取能量的能力
    - 每个社会成员每天能够摄取的食物、燃料和原材料的能力
  - 社会组织的能力
    - 一个社会里最大的永久性居住单位的人口数。
  - 信息技术的能力
    - 社会成员交流、储存、记忆的各种信息
  - 战争动员的能力
- 根据社会发展指数，对比东西方指数从公元前14000年-公园2000年，一直到公元前3000年东西方的发展几乎看不出任何差别，而公元前1800年以后呈现飞跃式发展
  + 蒸气机

### 农业文明的诞生

#### 幸运纬度带
  - 欧亚大陆北纬20度到北纬35度
    - 谷物（第二次文明的大跃进）
  - 美洲大陆北纬15度到南纬20度之间
  - 第一次文明的跃升：妇女的采集经验
  - 第二次文明的跃升：幸运纬度的农业
  - 农业：人类干预
#### 文明和文化
  - 文明：是用于描述人类发展的共性，并区别人类与动物的祖先
  - 文化：是指生活在不同地区的人们，在漫长的时间里形成的独特的生活方式、生活习惯及信仰
  - 工业文明开启了一个新的历史阶段；农业文明的到来也带来的一个新的历史阶段
    + 采集和狩猎 -> 农业文明 -> 工业文明（作者称为科技文明）
    + 不记得哪本书了，讲的最早的农业种植是因为要修建建筑，好像是祭祀的建筑（《枪炮、病菌与钢铁》）
#### 人类
  - 人类脑容量巨大，在出生时大脑还没有完全成熟，这跟其他的哺乳动物刚生产出来就可以站立，行走不同。
  - 人类成长周期长，成熟后比其他动物优势明显
  - 气候变化，人类开始走出非洲（4，5万年遍及全球）
    - 非洲索马里 -> 阿拉伯 -> 欧亚大陆 -> 北非 -> 欧洲 -> 亚洲 -> 澳洲 -> 阿拉斯加 -> 北美 -> 南美
  - 所有人类的共同祖先都是从非洲走出的智人的后代
  - 随着人类对全球的覆盖，人口的迅速增长，全球气候的变化人类有了第一次文明的飞跃
#### 农业
  - 人为改变自然，选择品种、施肥、除草
  - 畜牧业：人类驯化、圈养、配种、选择
  - 农业、畜牧业与人是相互储存的关系
  - 农业与畜牧业在全球分布的不均衡，最主要的原因是地理环境和自然资源完全不同
    - 地理环境在农业文明里起决定性作用
  - 中国的黄河流域是世界上自然资源第二好的地方，第一好的地方是西南亚，侧翼丘陵区；世界上的其他地区远远不如这两个地区
  - 美洲
    - 美洲的农业匮乏，美洲的玉米需要几十代的基因变种才有可能；美洲也没有能够驯养的动物；所以美洲的自然条件极其匮乏
    - 人类在公元前15000年通过大陆桥从欧亚大陆走到美洲大陆，而到了公园前12000年以后美洲和欧亚大陆被海洋分开；导致欧亚大陆出现的文明无法传播到美洲
  - 西南亚,侧翼丘陵区: 公元前9600年时出现农业
  - 中国：公元前7500年时出现农业
  - 西欧：公元前4500年时出现农业；虽然西欧的自然条件也很差，但交通相对通畅
  - 文明的传播方式：先进文明殖民落后的文明；另一种是落后文明向文明地区学习
  - 地理位置一方面决定了一个地区的自然禀赋，另一方面也决定了它和最先进文明的交流机会，由此造成了各个地区发展的差异
  - 东西方的概念：一个是西南亚和中东地区，一个是中国的黄河长江流域




### 农业文明的天花板及三次冲顶
- 直到工业革命到来之前，农业文明社会的发展轨迹始终遵循着“上升、冲顶、衰落”的循环规律；（中国古代的王朝更替）
- 农业文明一个天生不足的瓶颈：自然资源的上限
- 农业文明面临灾难的五个原因：饥荒、人口流动引起的战争、瘟疫、气候变化、政权失败
  - 在几千年的时间里，游牧民族和农业人口对土地的争夺一直都是战争的主要原因。
  - 农业人口之间的流动也是人口流动的一个主要源泉
  - 游牧民族的迁移给农业人口带来的另外一个直接结果就是细菌和病毒的传播，而以此引起的严重瘟疫，是历史上人口消减的最重要的原因
  - 先进政权和落后地区的差异也成为战争、资源掠夺和征服的一个主要原因
  - 叙利亚（大流士） -> 波斯帝国（色雷斯） -> 希腊 -> 罗马帝国
#### 三次顶峰
  - 在罗马帝国时期（秦、汉），社会发展第一次达到了农业文明的顶峰
    - 五大 挑战（前面讲的五个原因）几乎同时出现在东西方，引发了整个文明区域的毁灭性倒退
  - 唐宋，宋朝的东方帝国第二次冲到了农业文明的顶峰
    - 蒙古大军横扫整个欧亚大陆，征服了几乎所有文明中心的国家，也把瘟疫带到了世界的每一个角落
      + 蒙古大军的横扫欧亚大陆，将中国的技术发明传到欧洲，西欧没有受到蒙古的影响，只是把技术传到了西欧，而后面西欧发现了美洲，对美洲的殖民得以让西欧可以发展出工业文明
    - 中国当时的最重要技术发明，比如铸铁、火药、指南针、纺车、风车、水车、农业技术都传到了欧洲
      - 中国的火药传入后，迅速被发展成火枪和火炮
    - 蒙古大军到了匈牙利以后就戛然而止，完全没有到达西欧；没有破坏西欧，但技术传到了西欧；这为西欧成为下一次文明的爆发点提供了一个绝好的条件
  - 17、18世纪东西方都再次冲向农业文明的顶峰
    - 西欧因为中国技术的引进，出现新一轮社会发展
    - 马可·波罗对中国的盛赞，和寻找东方财富为下一次大航海运动提供了动力

### 农业文明中的思想革命与制度创新
#### 轴心时代
- 在公元前5世纪左右，出现了一次轴心思想革命，并命名为轴心时代
  - 东方中国：孔子开始讲述他的学说，与此同时诸子百家争鸣；（仁）
  - 西方中东：先知们开始把对世界、上帝的思考记录成《旧约圣经》（上帝）
  - 印度：释迦摩尼放弃皇子的优裕生活，开始和乞讨的人一起生活，和他们共同经历苦难，宣讲他解脱苦难的方法（涅槃）
  - 希腊：苏格拉底、柏拉图、亚里士多德，伟大的思想家都在全面地检测个人、社会、国家的意义（冥想）
  - **这些思想家几乎出现在同一个时期，来自欧亚大陆所有的文明中心，而且思想的指向惊人的一致**
    - 思考的问题都是文明毁灭之后普通人的痛苦、边缘人的挣扎、底层人的呻吟。
    - 关怀的对象都是弱势群体、普罗大众
    - 共同反对的是腐败、野蛮、欺瞒百姓的统治者和坏政府
#### 第二轴心时代（佛教、基督教、伊斯兰教）
  - 中国：**简化的佛教**被广泛传播
  - 西方：基督教成为罗马帝国的国教，迅速地在整个西方传播开来
  - 阿拉伯半岛的沙漠游牧民族：伊斯兰教出现
    - 创始人-默罕默德不识字，40岁时在梦里定期见到天使加百利给他传话
    - 经过他的传讲，吸引巨大的信徒群，在此后仅仅二三十年的余生中，他把一个沙漠边缘求生的、规模很小的游牧民族组织起来，征服了整个中东、埃及、地中海。他和他的后代创造了世界第三大宗教，建立了穆斯林帝国
    - 伊斯兰对所有拥有成熟文化的民族都表示出了足够的尊重、容忍和谦虚的学习态度
  + 现在的很多规则（价值）是由欧洲定义的
#### 科举制
- 轴心时代思想最重要的遗产是高端政府政治制度的建设
  - 中国的轴心时代思想直接导致了中国历史上最伟大的政治制度创新--科举制
    - **科举制是整个农业文明时代最伟大的制度创新**
    - 管理政权
- 科举制从社会各阶层选拔人才，凭考核结果分享政治权力，**另外通过政府考试的试统一官方意识形态**
  - 从思想意识形态上，士大夫既效忠皇权政统，又追随儒家道统
- 科举制起源于两汉时的荐贤尝试，到隋朝正式确立成为制度，为此后一两千年里管理中国提供了最坚实的保障
  - 解决了作为庞大帝国的行政问题
  - 保障了长期的和平环境
  - 形成了大规模的贸易市场，促进了技术交换和广泛应用
  - 发展了文化拥有了应对饥瘟疫、外族侵略的能力
  - 使中国在之后的1000多年里领先于世界上几乎任何其他国家
- **大英帝国开始建立的时候，也借鉴了中国的文官制度；无论是美国的军队，还是其他采用文官系统的政府或是非政府组织都多少受到了中国科举制度的影响**
- 核心缺点：
  - 皇帝，以血缘传承，但血缘无法保障能力
    - 不稳定的皇权传承影响着朝代的兴衰

### 美洲大陆的发现及其划时代的影响
#### 大航海时代
  - 葡萄牙-达迦马：绕过非洲好望角建立
  - 哥伦布出海：绕过大西洋直接寻找亚洲，结果发现美洲
    - 欧洲人到美洲时，只有墨西哥和南美两个比较初级的政权
    - 欧洲人最厉害的武器还不是铸铁和枪炮，也不是战斗能力，而是他们身上携带的病菌
      - 黑死病和天花让活下来的欧洲人身上已经有了抗体，但是这些细菌并没有消失，一直和人畜并存着；
    - 哥伦而到达美洲之后，只用了几代人的时间，就让75%的美洲当地人丧生于细菌
    - 欧洲人在16世纪初期发现自己继承了一个崭新的大陆，新大陆的自然条件非常好，**适合种植农作物，发展畜牧业**。
    - **美洲的面积是西欧的将近9倍，还拥有丰富的自然资源，比如白银及其他矿产资源**；
    - **美洲彻底改变了欧洲的经济状况，还解决了西欧的土地瓶颈**
    - 欧洲人用少数奢侈品到非洲去换来奴隶，让奴隶在美洲种植蔗糖、棉花、树木，再把农产品支架欧洲，把新的工业品运往美洲，形成了一个巨大的、环大西洋的贸易圈。**为欧洲经济突破农业经济瓶颈创造了条件**；
    - 西班牙与葡萄牙不重商，只把商人当做提款机
      + 所以西班牙与葡萄牙虽然最早开始大航海，但没落的很快
    - 荷兰和英国
      - 英国从1215年《大宪章》开始一直在削弱君权，此后议会不断从皇权中分权
      - 15世纪以后，任何有一定数量财产的公民都可以通过选举成为国会的下议院成员，所以下议院逐渐成为成功商人的代言
      - 17世纪，下议院与国王内战，今天的议会制初步建立
      - 1688-1689：经过政变，一位荷兰王子坐上了名义上的皇位，并签署了《权利法案》，标志着人类历史上第一个君主立宪有限政府的成立，第一个重商的宪政国家的出现。这个政府的权力主要是在下议院
#### 现代科学革命
  - 美洲新大陆的发现和航海时代的到来，这个时代的人们开始面临一些最根本的技术问题，无论是对地理学、地质学、生物学、航海技术、天文学，还是对政府的，经济的本质等等；
    + 这个时代的西欧，英国开始了君主立宪，是更重视商人
  - 1687年牛顿出掉的《自然哲学的数学原理》开始了一场现代科学革命，带给欧洲人一种全新的世界观，并开启了一个全新的时代
  - **环太平洋的新型的自由市场经济和科学革命一起，为现代化的诞生提供了根本的条件**
    + 美洲的发现，对西欧的发展产生的影响，这里是关键拉开了西欧与东方的差距

### 现代化的诞生
- 1776年，看起来毫不相干的三件事发生了
  - 亚当·斯密在英国出版了《国富论》
    - 《国富论》主要讨论的核心问题是大西洋经济的本质
    - 美洲是完全无政府的状态
    - 英国商人因为大西洋经济而迅速地成为有力量的社会成员
    - 这个新经济实质就是在英国、荷兰和大西洋另一端的北美洲，形成了一个环大西洋的，以**有限政府、完全资本主导**的自由贸易的经济
    - 自由市场-看不见的手
    - 政府对自由经济的合适角色就是不干预、不作为，政府的主要功能是保护私人财产，保证自由竞争存在，反对垄断，保证自由市场的笔译，在国际间推动自由贸易
    - 亚当·斯密的后继者李嘉图基于对**社会分工**的分析进一步阐述了自由贸易的优势
      - 贸易带来了繁荣和财富，并且市场越大，贸易带来的财富增量也越大
    - 由于亚当·斯密的理论对英美形成了长期持续性的影响。英美两国的政府功能，尤其是对殖民地的态度，和其他殖民地的强权表现出了很大的不同（西班牙、葡萄牙）。
  - 美国独立，美国国父们发表了《独立宣言》
    - 美国深受
  - 瓦特在伯明翰家宣布制造了世界上第一台蒸汽机
  - 这三件事的联合，使这一年成为了人类文明的分水岭，从此人类文明再次跃升了一个新的阶段
  - **资本主义**就像亚当·茂密预测的那样，最终造福了几乎所有的人：资本家、劳工、生产资源所有者、消费者
+ 历史上有很多巧合
  - 前几章讲的农业文明的思想革命；孔子，释迦摩尼欧洲的先知们是同一个时期的人物
+ 由西欧的大航海发现的北美洲，从而形成的**自由贸易**，英美两国政府成为自由贸易，自由 市场的全球推动
  - 自由贸易创造出的财富，带来的问题产生了工业革命
- 政治经济学家从**劳动力**来解释价值的创造，其中最有影响的是**马克思**
  - **人的劳动最终创造了一切价值，但是劳动的果实不公平的被资本家剥削，预测这种剥削最终会导致资本主义的经济危机，让世界进入到一种新的形态，即共产主义**
