# 学缠论重点中的重点就三个：中枢、级别、走势

# 级别
  - 次级别为当前级别的1/6（如60分钟的次级别是5分钟）
  - 顶分型相当于当前级别的次级别向下一笔，底分型相当于当前级别的次级别的向上一笔
# K线包含合并
  - 处理包含关系时，趋势方向的判定原则：
    + 1、当遇到第一根需要处理的 K 线时，先看前二根 K 线，这两根 K 线的方向决定了之后采用 GG 还是 DD 原则；
    + 2、若之前两根 K 线是上升趋势，则采用 GG 原则，反之，采用 DD 原则。
  - 处理包含后再包含的原则是：按照时间顺序，相邻两根发生包含就处理。
# 分型
  - 顶分型
  - 底分型
  - 没有顶分出现就没有顶，没有底分出现就没有底
  - 中继、转折顶底分型的区分
    - 分型停顿
      - 当分型形成后，等待后续某一根K线的收盘价站住分型的第三根K线的极值
      - 可以帮助判断是否是中继分型，实质是次级别还有结构没有走完。当前级别停顿一两根K线，次级别就有12根K线走完，大部分的次级别结构也可以判断清楚
    - 观察次级别走势
  - 验证分型
# 笔
  - 向上笔
  - 向下笔
# 线段
  - 至少需要3笔，且三笔必须有重叠的部分
  - 向上线段
  - 向下线段
  - 线段破坏：线段只能被另一个线段破坏
  - 线段结束：
    - 特征序列：向上线段中，向下笔为该线段特征序列；向下线段中，向上笔为线段的特征序列；
      - 分析时可以将特征序列的笔看做是一根根K线，然后判断是否有顶底分型，顶底分型出现意味着线段的终结
# 中枢
  - 某级别走势类型中，被**连续前三笔所重叠**的部分
    - 高点取低的，低点取高的
  - 本级别画不出笔的震荡区间，由小级别延伸而成
  - 中枢的作用
    - 中枢的引力作用
    - 压力支撑作用
  - 次级别中枢：本级别图中看到横向8根及8根以上，有相互重叠区间
  - 次次级别中枢：本级别图中看到横向4根~8根，有互相重叠区间
  - 中枢在没有被第三类买卖点破坏之前一直有效；一旦被破坏掉，则立即放弃
# 背驰
  - 没有新高、新低就没有背驰
  - 盘整背驰（盘整背驰未经过中枢）
  - 背驰（背驰经过中枢）
  - 判断是否背驰：
    - 结合MACD指标，如果没有回抽过零轴，通常不是本级别真正意义的背驰，可以只是次级别或次次级别的背驰
      - MACD快慢线回抽0轴
      - MACD红绿柱面积大小
    - 力度：长度及速度的结合，通常可以用长度代替
      - 判断力度大小：可以参考MACD指标的黄白线及红绿柱的高度，以及红绿柱的面积
  - 背驰段
    - 区间套，不断的在小一级别图中查找背驰段，可以更精确的定位
# 买卖点
  ## 一买：为下跌中枢下方，下跌力度较进入中枢段衰竭而形成的买点
    - 两种情况下一买的成功率高一些：
      - 二中枢背驰
      - 力量悬殊的一中枢背驰
    - 一买大概率可以反弹回最近一个中枢，但一中枢背驰后有可能做第三类卖点
    - 一买失败原因：
      - 买在中继底分型，形成下跌中枢延伸
      - 下跌背驰后以另一个下跌中枢来替代上涨
  ## 二买：属于多空双方力量逆转的买点
    - 在第一类买点后第一次空头反抽不创新低的点
    - 使用频率最高的买点（回抽零轴）
    - 一买涨幅过大会影响二买的上涨空间
    - 二买定位：
      - 任何买点本质上都是一买，将形成二买的下降笔放大到次级别，再使用一买的方式买入
      - 在次级别使用底分型停顿
    - 二买处理小转大走势
    - **止损上移**
      - 如果一买建仓，二买补仓的话，原本设置的在一买位置的止损点上移至二买位置
  ### 类二买：多方蓄势待发的买点
    - 与二买一起构成上涨中枢
  ## 三买：属于乘胜追击的买点
    - 在离开第一个上涨中枢后，第一次空头反拉不进中枢的点
  - 如何衡量买点的合理性--R比率
  - 如何衡量选股水平的高低--风险收益比
  ## 一卖：在上涨中枢上方，由于上涨力度较进入中枢段衰竭而形成的卖点
    - 卖出后接下来行情不再背驰，注意回补，当回调至3买价格附近时
  ## 二卖：一卖出现后多方第一次反扑无法创新高的点
    - 能卖在一卖绝不卖在二卖
  ## 三卖：是针对最后一个上涨中枢回拉不进中枢的点，或者是第一个下跌中枢回拉不进中枢的点
    - 绝对不能使用三卖来卖出股票

# 选股
  ## 一买选股
    - 1、海选：
      - 30分钟级别：使用20日跌幅排序；跌幅大于15%
      - 日线级别：使用年跌幅排序；跌幅大于30%
    - 2、精选：
      - 30分钟：保留下来日线至少回抽10日均线的个股（回抽最多不能超过20日均线）
      - 日线级别：保留下来周线至少回抽10周均线（注意，回抽10日均线时剔除高级别已成笔的个股）
    - 3、淘汰：
      - 看下跌之前的股性，只保留下跌前活跃的，上涨有力的
      - 最优为两中枢背驰，其次为一中枢悬殊背驰
  ### 一买步骤：
      - 1、选操作级别
      - 2、寻找背驰段
      - 3、在背驰段创出新低后，开始寻找典型的底分型
        - 停顿法
        - 次级别是否背驰
      - 4、买入
      - 5、设置离场规则
        - 1）、底分型中价格最低点
        - 2）、底分型最后一根K线最低点
  ## 类二买选股
    - 在月线上寻找横向回抽20月均线的个股，意味着在月线的次级别，也就是周线很有可能是类二买
    - 在周线上寻找横向回抽20周均线的个股，意味着在周线的次级别，也就是日线很有可能是类二买
    - 优选类二买的原则
      - 板块效应：板块指数，同版块中是否有多个股票处在类二买位置
      - 均线辅助：向上一个级别10日及20日均线的支撑
      - 大小级别叠加：类二买+类二买、类二买+三买
  ## 三买选股
    - 30/60分钟级别：5日涨跌幅大于15%，20日涨跌幅大于-5%，小于15%；（扩大范围，5日可改为10日）
      - 最好有涨停板
    - 日线级别：20日涨跌大于40%
    - 多翻热门板块个股
  ## 高白股选股 （经过放量猛涨的个股，活跃度大幅增加）
    - 题材：
      - 阶段性热门板块龙头股
      - 因为重组、借壳等题材出现连续涨停的个股
    - 走势：
      - 连续3个涨停板以上
      - 20日涨跌幅大于60%
    - 排除掉日线级别已经走出两个中枢的个股
    - 操作：
      - 30/60分钟级别
      - 出第一个上涨中枢的快速上涨中，在3买位置，回调一笔考虑短线介入
        - 第一个中枢可以反复操作
        - 若是第二个中枢关注是否背驰，背驰换股
# 自选股的跟踪与更新
  - 涨速：系统预警，股票拉升前通常会有涨速异动
  - 量比：
    - 早晨：9：30~9：35 量比大于3的个股当日异动概率高
    - 收盘后：量比大于1的个股后续异动率高
  - 更新：随时剔除已经走坏或已经有大幅涨幅的股票
  - 卖错后回补：本级别未背驰，在本级别一笔下跌后进行回补
